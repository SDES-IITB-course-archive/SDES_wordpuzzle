List of Modules
===============

We have followed Model–view–controller (MVC) software architectural pattern for implementing user interfaces. It divides a given software application into three interconnected parts, so as to separate internal representations of information from the ways that information is presented to or accepted from the user.

See for more detailes http://en.wikipedia.org/wiki/Model%E2%80%93view%E2%80%93controller
